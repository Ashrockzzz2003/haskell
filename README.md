# Haskell

## Installation

### MacOS

```bash
curl --proto '=https' --tlsv1.2 -sSf https://get-ghcup.haskell.org | sh
```

## Running the code

### Option 1 (Running in GHCI) (No main function needed)

```bash
ghci

# In GHCI terminal

:load file_path

# Function now loaded and is ready to use.
```

### Option 2 (Compile and execute) (Main Function needed)

#### Compile
```bash
ghc <file_name>.hs
```

#### Run
```bash
./<file_name>
```
